# Basic-Calculator
A basic calculator for use in the Python terminal
## How Does It Work?
The program starts by asking the user which language (English/Turkish) they prefer to use in the Python terminal,then asks the user for two different integers.
The two numbers entered are calculated using one of the four mathematical operations preferred by the user, and the answer is given to the user.
## Error Messages;
If the user does something that causes the code to be incomplete or malfunction while using the program,<br>
the program displays an error message and stops running if necessary.
## How Can It Be Improved
+Additional languages can be added to the language selection section or removed entirely. <br>
+The program can be set to start a new operation after completing each mathematical operation instead of closing. <br>
+The program can be set to save each mathematical operation as history after performing it. <br>
+New and more complex mathematical operations can be added to the program.
